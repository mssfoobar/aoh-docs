🔩 Technologies

The list of notable technologies used in the project.

📄️ Svelte & SvelteKit

We've chosen Svelte as our primary component-based UI framework and SvelteKit as the web application framework

📄️ Tailwind CSS

Tailwind is a utility-first CSS framework that we use for rapidly building custom user interfaces.

📄️ Playwright

We use Playwright to automate testing on our web app.

📄️ Docker

We use Docker to run our containers.

📄️ Pino

Pino is a very low overhead Node.js logger that we use to provide structured logging.

📄️ Font Awesome

We've chosen Font Awesome to be our primary icon library.

📄️ Apache ECharts

We use Apache ECharts as our primary charting library.

📄️ MapBox

MapBox is a library we use to render dynamic maps.

📄️ urql

urql is a highly customizable and versatile GraphQL client

📄️ TypeScript

We use TypeScript on top of JavaScript to help our code be much more maintainable.

📄️ GraphQL

GraphQL is a query language for that we use for our APIs.

📄️ Docusaurus

We use Docusaurus to generate our documentation from Markdown files.

📄️ Faker

We use Faker to generate fake (but realistic) data for testing and development.

📄️ Others

A list of less critical dependencies used in AgilRad 2.0 Web.
