  • Git

Configuration​

AWS account and profile

To set profile:

For windows:

setx AWS_PROFILE <clustername>
aws eks update-kubeconfig --region ap-southeast-1  --name <clustername>

For linux:

export AWS_PROFILE=<clustername>
aws eks update-kubeconfig --region ap-southeast-1  --name <clustername>

Log into your aws account. Check that your ".aws" folder is populated with the correct credential.

top level domain​

Prepare a top level domain to be used by AOH. The AOH applications requires an endpoint to be reachable from the internet to function. For example: *.wfm-alpha.agilrad.com
