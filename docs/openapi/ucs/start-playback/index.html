Start playback​

API to start playback

Path Parameters
    playbackid string required

    ID of playback room

    Example: df91094f-1fdd-4be6-a8c6-e4038ccc87f1
application/json
Request Body
    speed integer

    speed0: double rounded down to 1 decimal place
    speed1: fast motion is > 1.0, slow motion is < 1.0
    speed2: default is 1.0 (as recorded speed)

    playfrom integer

    number of seconds from start of recording, default is play from current state

    chat boolean

    boolean to playback chat, default is true

    video boolean

    boolean to playback video, default is true

    audio boolean

    boolean to playback audio, default is true

Responses
  • 200
  • 400
  • 500

successful operation

Bad Request

Internal Server Error
