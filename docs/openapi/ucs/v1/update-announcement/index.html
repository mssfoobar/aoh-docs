Update announcement

PUT 
/v1/rooms/:roomid/announcements

API to overwrite announcements by id

Request​

Path Parameters

    roomid stringrequired

    ID of room to update announcement

    Example: f81d4fae-7dec-11d0-a765-00a0c91e6bf6
  • application/json

Body

    requestor string

    Optional field of user_id calling the API

    announcements

    object[]

  • Array [

    • id string

    Optional & Unique field

    message string

    Optional field message field

    relative_from string

    default is 'end'. accepts only NULL|'start'|'end'

    relative_time_in_seconds integer

    Optional field

    user_id string[]

    default is messsage to all

  • ]

Responses​

  • 200
  • 400
  • 500
