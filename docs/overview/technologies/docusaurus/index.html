Docusaurus

We use Docusaurus to generate our documentation from Markdown files.

What is Docusaurus?​

Docusaurus is a React-based static site generator from Facebook. It creates documentation sites from Markdown files and provides many convenient rendering options, theming, as well as versioning and search.

Why we've chosen Docusaurus​

Docusaurus is open source, easy to pick up, and looks great. It is extremely popular and is used for documentation in many open source projects.
